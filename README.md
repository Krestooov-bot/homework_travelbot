# ✈️ TravelBot: AI Планувальник подорожей

## 📖 Опис проєкту
Цей проєкт реалізовано в рамках завдання "Варіант 2. Планувальник подорожей «TravelBot»". Це інтерактивний веб-застосунок для планування подорожей, який допомагає користувачам створювати маршрути, автоматично підраховувати бюджет та зберігати список місць для відвідування.
